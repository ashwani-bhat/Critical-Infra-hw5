1) Install the following Libraries using pip:
   a) Numpy (pip install numpy)
   b) Pandas (pip install pandas)
   c) Sklearn (pip install sklearn)
   
2) The Data(.xlt) is converted into .csv. This csv file should be read in the code(already included 'HW_TESLA.csv' in folder)

3) Provide your test input in 'test.csv'.Already provided in folder(Initially contain column name same as that mentioned in 'HW_TESLA.csv').

4) Change current working Directory to this folder.

5) To Run Decision Tree code(Best Model), Run 'py -3 dt_submit.py' using cmd
   
6) To Run SVM(with PCA) code, Run 'py -3 svm_submit.py' using cmd
